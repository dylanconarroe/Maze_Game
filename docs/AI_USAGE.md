# AI-Assisted Development

This project used AI as a development support tool while all code was reviewed, tested, and manually verified.

## How AI Was Used

| Area | AI Contribution | Verification |
|---|---|---|
| Project planning | Helped brainstorm a portfolio-friendly maze game concept | Selected features based on scope and learning goals |
| Maze generation | Explained recursive backtracking and dead-end placement strategies | Tested generated mazes manually and with unit tests |
| Pathfinding | Helped review BFS solver logic | Verified paths with automated tests |
| Testing | Suggested edge cases for movement, maze generation, and pathfinding | Converted useful cases into Vitest tests |
| Documentation | Helped draft README and project explanation | Edited and reviewed final wording |

## Responsible Use

AI was used to support planning, debugging, and documentation. I manually reviewed the code, tested the game behavior, and used automated tests to verify key logic.
