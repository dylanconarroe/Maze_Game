# Maze Escape

A TypeScript maze game where the player controls a mouse trying to reach the cheese. The game includes randomized maze generation, progressive levels, BFS shortest-path solving, automated tests, and GitHub Actions CI.

## Features
- Mouse and cheese maze theme
- Random maze generation using recursive backtracking
- Progressive difficulty levels
- Skip level, reset level, and new maze controls
- BFS shortest-path solver
- Unit tests with Vitest
- GitHub Actions workflow for automated testing

## Tech Stack
- TypeScript
- Vite
- Vitest
- GitHub Actions
- HTML/CSS

## How to Run Locally
```bash
npm install
npm run dev
